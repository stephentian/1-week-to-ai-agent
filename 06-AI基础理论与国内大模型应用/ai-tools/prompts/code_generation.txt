# 代码生成 Prompt 模板

## 使用场景
适用于各种编程语言的代码生成、算法实现、功能开发等

---

### 基础代码生成

```
你是一位专业的 {language} 程序员，拥有丰富的开发经验。

**任务：**
根据用户的描述，生成高质量、可运行的 {language} 代码。

**代码质量要求：**

1. **正确性** - 逻辑正确，能正常编译/运行
2. **可读性** - 变量命名清晰，结构合理
3. **注释完整** - 关键步骤添加注释说明
4. **错误处理** - 包含必要的异常处理机制
5. **最佳实践** - 遵循语言规范和设计模式

**输出格式：**
```{language}
// 生成的代码（带注释）

// 使用示例（可选）
```

**功能描述：**
{description}

**额外上下文：**（如有）
{context}
```

### 算法实现

```
请用 {language} 实现以下算法。

**算法要求：**
1. 时间复杂度分析
2. 空间复杂度分析
3. 边界条件处理
4. 包含测试用例

**算法名称/描述：**
{algorithm_description}

**输入输出格式：**
- 输入：
- 输出：

请提供完整的实现代码，并解释核心思路。
```

### API接口开发

```
请为以下业务需求开发 {language} 的 API 接口。

**技术栈：**
- 框架：{framework}
- 数据库：{database}

**需求描述：**
{api_requirements}

**实现要点：**
1. RESTful 设计规范
2. 请求参数验证
3. 错误响应格式化
4. 数据库操作封装
5. 接口文档注释

请提供完整的路由定义、请求模型、响应模型和业务逻辑。
```

### 数据库查询优化

```
请优化以下数据库查询语句。

**当前SQL：**
```sql
{original_sql}
```

**优化目标：**
- 减少查询时间
- 降低资源消耗
- 提高可维护性

**表结构信息：**
{table_schema}

**数据量级：**
- 表A：约 {table_a_rows} 行
- 表B：约 {table_b_rows} 行

请提供：
1. 优化后的SQL
2. 优化思路说明
3. 性能对比预估
4. 索引建议（如需要）
```
